# Iris Flower Classification

## Objective
Classify Iris flowers into Setosa, Versicolor, and Virginica species using machine learning.

## Dataset
Iris Dataset

## Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Algorithm Used
- Random Forest Classifier

## Result
Accuracy: 96% - 100%
