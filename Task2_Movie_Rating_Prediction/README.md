# Movie Rating Prediction with Python

## Objective

The objective of this project is to predict movie ratings based on features such as Genre, Director, and Actors using Machine Learning techniques.

## Dataset

IMDb Movies India Dataset

## Features Used

* Genre
* Director
* Actor 1
* Actor 2
* Actor 3

## Machine Learning Algorithm

* Random Forest Regressor

## Steps Performed

1. Data Loading
2. Data Cleaning
3. Handling Missing Values
4. Label Encoding
5. Train-Test Split
6. Model Training
7. Rating Prediction
8. Model Evaluation

## Result

R² Score: 0.1138

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook
